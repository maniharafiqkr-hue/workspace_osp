# OSP Week 1 Practice
My first project for the Open Source Programming course.

## How to run
1. Install Flask:
   pip install flask
2. Run the server:
   python -m flask run
3. Open http://127.0.0.1:5000 in your browser.

- app.py: Main Flask application
